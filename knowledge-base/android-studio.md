Go from idea to app

With the New Project flow, you can describe your vision in natural language or even upload a UI mockup and let Gemini (or your LLM of choice via API key) build the foundation for you instead of spending hours wiring up dependencies.

How it works:

1.	Open the New Project Wizard and select "Create with AI".

2.	Describe your app's flow (e.g., "A workout tracker with a calendar view and local storage"). You can even upload images for style inspiration.

3.	The agent creates a project plan, writes the Kotlin and Compose code, resolves build errors autonomously, and deploys a working draft to your emulator.

Version Upgrade Assistant



Maintenance is now as fast as creation. The new Version Upgrade Assistant handles the manual slog of dependency management. It iteratively updates your libraries, reads build errors, and adjusts versions until your project is stable, so you don't have to.




Pro-Tip: Level up with a Gemini API key
While these features work out-of-the-box, connecting a paid Gemini API key unlocks Nano Banana for high-fidelity UI generation when using the New Project flow, and Gemini 3.1 Pro for agentic coding. Note, if your business requires using another LLM, you can bring your own API key.
